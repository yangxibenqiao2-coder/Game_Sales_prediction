# Game_Sales_prediction

## 概要

ゲームソフトの属性情報を用いて、販売本数を予測する回帰モデルを構築したプロジェクトです。

学内コンペ形式で行われ、評価指標は決定係数 R² です。

## 目的

発売年、ジャンル、プラットフォーム、出版社、タイトル情報などから、

ゲームの総販売本数を予測するモデルを作成することを目的としました。

## 使用技術

- Python

- pandas / numpy

- scikit-learn

- LightGBM

- TF-IDF

- Target Encoding

## 工夫した点

- 2000年以前のデータを除外し、市場構造の違いによるノイズを軽減

- 地域別売上はリークの可能性があるため除外

- publisher / developer / console / genre などに Target Encoding を適用

- タイトル文字列に対して TF-IDF を適用

- 年別発売本数、タイトル長、数字の有無などの特徴量を作成

- LightGBM のハイパーパラメータを調整

## モデル

最終的に LightGBM Regressor を採用しました。

非線形な関係やカテゴリ変数由来の複雑な相互作用を扱いやすく、

線形回帰よりも高い精度が得られたためです。

## 結果

- Validation R²: 0.6546

- MAE: 約0.129

- RMSE: 約0.207

## 学び

構造化データでは、モデル選択以上に前処理と特徴量設計が精度に大きく影響することを学びました。

特に、カテゴリ変数の扱い、タイトル情報の活用、リークを避けた特徴量設計の重要性を理解しました。
